Rabbit Blast

A quick block puzzle game for short sessions and score chasing.

About the game

Place colorful blocks on an 8×8 board, clear rows and columns, build combos, and challenge your personal best. Magical boosters can help open space when a run gets tricky.

Rabbit Blast is a single-player game. The current version does not require an account, cloud save, chat, or in-app purchases.
