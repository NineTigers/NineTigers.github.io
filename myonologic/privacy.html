2. 기본 개인정보 처리 원칙

묘노로직 앱은 계정 가입, 로그인, 자체 회원 서버, 결제 기능을 제공하지 않습니다. 게임 진행과 설정 정보는 기본적으로 사용자의 기기 안에 저장됩니다.

앱 버전에 따라 광고 SDK 사용 여부가 다릅니다. 1.0 이하 버전은 광고 SDK를 사용하지 않으며, 1.1 이상 버전은 광고 제공을 위해 Google AdMob / Google Mobile Ads SDK를 사용할 수 있습니다.

3. 기기 내 저장 정보

앱은 게임 진행과 설정을 위해 다음 정보를 사용자의 기기 안에 저장할 수 있습니다.

  • 클리어한 스테이지
  • 선택한 난이도와 스테이지
  • 생명력 및 회복 타이머 상태
  • 언어 설정
  • 음악, 효과음, 진동 설정

이 정보는 운영자 서버로 전송되지 않으며, 사용자의 기기 안에서만 사용됩니다.

4. 광고 및 제3자 SDK

1.1 이상 버전은 보상형 광고, 배너 광고, 전면 광고 제공을 위해 Google AdMob / Google Mobile Ads SDK를 사용할 수 있습니다. Google Mobile Ads SDK는 광고 제공, 광고 빈도 제한, 부정 사용 방지, 광고 성과 측정 등을 위해 광고 식별자, 기기 정보, IP 주소, 앱 정보, 광고 상호작용, 진단 정보 등을 처리할 수 있습니다.

현재 앱은 자체 분석, 자체 크래시 리포팅, 로그인, 위치, 푸시 알림, 고객지원 채팅 SDK를 사용하지 않습니다.

5. 광고 및 추적

현재 구현은 비개인화 광고 요청을 사용합니다. 향후 맞춤형 광고, 리타게팅, IDFA 또는 광고 ID 기반 추적을 활성화하는 경우, 앱 권한 요청과 스토어 개인정보 선언을 별도로 업데이트합니다.

6. 보관 및 삭제

기기 안에 저장된 게임 진행과 설정 정보는 앱 삭제 시 함께 삭제될 수 있습니다. 서버에 저장된 계정 정보가 없으므로 별도의 계정 삭제 절차는 제공하지 않습니다.

7. 변경

앱 기능, 수집 정보, SDK, 서버 사용 방식이 변경되는 경우 이 개인정보처리방침을 업데이트합니다. 중요한 변경이 있는 경우 앱 또는 배포 채널을 통해 알릴 수 있습니다.
